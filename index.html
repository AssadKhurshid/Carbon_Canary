Methodology

Built on verified EU standards

🇪🇺
EU JRC Database
Every emission factor sourced from the EU Joint Research Centre's EPLCA database. Each value includes publication year and document reference.
📖
IPCC AR6
Global Warming Potential values use IPCC Sixth Assessment Report (AR6) 100-year GWP factors — the current gold standard for corporate carbon accounting.
✅
CSRD-Aligned
Output follows GHG Protocol Scope 3 Standard, aligned with ESRS E1 requirements of the EU Corporate Sustainability Reporting Directive.
